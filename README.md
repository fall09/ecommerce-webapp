# E-Commerce Backend

This project is the backend service of an E-Commerce application built with Spring Boot.

It provides REST APIs for:

- Authentication (Login / Register)
- Products
- Categories
- Cart
- Orders

--------------------------------------------------

FRONTEND REPOSITORY

The frontend of this project is developed with React + Vite.

Frontend repository link:
https://github.com/fall09/ecom-frontend

--------------------------------------------------

TECHNOLOGIES

- Java
- Spring Boot
- Spring Security
- JPA / Hibernate
- PostgreSQL
- Docker
- Maven

--------------------------------------------------

RUNNING THE PROJECT

Start the backend server with:

./mvnw spring-boot:run

or

mvn spring-boot:run


Backend will run on:

http://localhost:8080

--------------------------------------------------

FRONTEND

Make sure the frontend project is running on:

http://localhost:5173

--------------------------------------------------

API BASE URL

http://localhost:8080/api
